What is Cave Test?

- Cave Test is a Minecraft clone developed using technology from 1993. it is currently modeled after Minecraft Pre-Classic, the earliest versions of Minecraft.

- It is developed using Open Watcom and tested on DOSBox, an MS-DOS emulator.

- DOS/4G was a 32-bit extender developed in 1993 by Rational Systems, and DOS/4GW was a free edition packaged with the original Watcom C Compiler.

- This project originated as a thought experiment after seeing The Science Elf on YouTube run a minecraft server on a computer from 1965. How early could a game such as Minecraft have been developed, given the technological constraints of earlier eras?

- The current version of Cave test is ct-212145, given the same versioning style as Minecraft version rd-132211 (ddhhmm).

- The function is very similar to Minecraft Pre-Classic versions. This version includes movement controls, and breaking and placing blocks. Four blocks are available, selectable with the 1-4 keys and can be placed on a 32x32x24 world map. This version includes simple shadow rendering along with independent textures for each block face.

How To download and play Cave Test:

- First download DOSBox for your operating system. This is an MS-DOS emulator, not a virtual machine.

- Download the Cave Test exe (cave_test.exe) and dos4gw.exe. Put these executables in the same folder.

- In DOSBox, type "mount C C:\(filepath to your folder)".

- Switch to the C: drive by typing "C:"

- type "cave_test.exe" to run the program. You may need to click on the window to access the in-game cursor.

- Keep in mind that a known bug occurs when attempting to break or place outside of the 5 block range. This bug disables the ability to break or place and the program must be restarted.

- to quit, press Esc. To get your cursor back, you may need to press Ctrl+Alt+Del and Cancel. You may also type "exit" if you are completely finished. You will need to re-mount the C: drive every time you re-start DOSBox.
